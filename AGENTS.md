When it comes to this project! I need you to remember these!

1. When I ask you to change syntax of Mace always update the docs, parser and zed extension
2. If the Tree Sitter parser is changed always update `zed-extension-mace/extension.toml` `grammars.mace.rev ` with the latest commit
3. Please push changes to the parser after every commit!
